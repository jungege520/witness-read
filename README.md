witness-read(阅读见证开源项目)
================
前端基于iview + vuejs2.0 构建，后端基于lumen5.4 + jwt + dingo，使用的前后端分离的模式开发。[测试地址](https://read.fastgoo.net/vue-admin/dist/index.html)  账号 admin  密码 123456
